# Symfony Playground "one"

* Symfony 5.1.*
* Docker compose file for PostgreSQL
* dev login form - prod google login
* JQuery, Bootstrap and Bootswatch
* Prepared for Heroku

## Usage

1. `composer install`
1. `yarn`
1. `yarn dev`
1. `bin/start`

Use `symfony console user-admin` to create an admin user.

Happy coding `=;)`
